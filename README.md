## <SysManAngular > 一款基于 Angular 的 PC 端后台管理系统界面
### 1. 技术介绍
    该系统基于angular-1.5.7，构建工具为gulp,我使用的是Hbulider编辑器。路由的部分用的是angular-route，
    相对于ui-router，它功能有一定的局限性，无法做到多层的子路由。符合合同银行业
### 2. 项目构建方法
    本项目基于 gulp 构建。
    第一步，你需要安装 nodejs 运行环境（有关工具的安装和使用办法请自行 google），全局安装 gulp ，
    再在 pull 下来的文件夹内，安装本地 gulp ,npm i 下载开发依赖，等下载完毕，打开浏览器输入 
    localhost:1234 就可以看到项目了。因为项目中用到了热更新，所以只要你一直开着 gulp 运行环境，那么
    你每次修改了代码，浏览器都会自动刷新。
